# C LOOPS
- **Loops are  used to repeat a block of code in a computer **
  ### There are three types of loop on c 
### while loop 
``` c 
int x = 0; 
while(x<5){
printf("%d \n ",x);
x++;
}
```

### Do while loop  
- This code first run and cheek the conditions 
``` c
int x = 0;
do{
	printf("%d",x);
	x++;
} while(x<5);
```

### For loop 
``` c 
int x = 0 ;
for(x=0;x<5;x++){
printf("hello %d",x) ;
}
```
